Cambridge_FaceDB contains sample images of the faces available at http://www.cl.cam.ac.uk/research/dtg/attarchive/facedatabase.html

testcase.txt contains labels and image file path for each test cases.

traincase.txt contains labels and image filepath for each sample cases which needs to be trained.


How To Use:

To train the sample faces from traincase.txt
./OnlineFaceRec train traincase.txt

To test the sample test faces from testcase.txt
./OnlineFaceRec test testcase.txt
